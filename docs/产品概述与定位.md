# RAG 企业知识库平台 — 产品概述与定位

> 面向产品经理、决策者、技术负责人。5 分钟掌握产品全貌。

---

## 一句话定位

面向中小技术团队的内部知识库问答平台——上传团队文档，自然语言提问，基于文档内容精确回答并标注引用来源。

---

## 核心价值

| 痛点 | 解决方案 |
|------|---------|
| 团队文档散落在各个角落，找不到想要的内容 | 统一上传到知识库，检索+AI 回答一步到位 |
| 研发文档（方案/接口/规范）靠口口相传 | 上传后新人可直接提问，答案有来源可追溯 |
| 已有文档但没人看、翻不动 | 自然语言理解，跨文档搜索，自动提取关键信息 |

---

## 使用场景

1. **技术文档问答**：上传架构方案、API 设计文档、技术规范 → 团队成员问"xx 模块的接口怎么定义的"直接得到答案
2. **新人入职培训**：上传团队文档 → 新人自助提问，减少老员工答疑负担
3. **项目知识沉淀**：项目复盘/会议纪要/技术决策记录上传 → 后续查询可追溯

## 功能全景

### 知识库管理
- 创建多个独立知识库，按团队/项目隔离
- 支持成员管理：owner 可添加/移除成员
- 管理员可查看和强制删除任意知识库

### 文档处理
- 支持 **6 种格式**：PDF、Word、Markdown、TXT、Excel、PPT
- 上传即后台异步处理（不阻塞用户操作）
- 自动解析→智能切分→向量化→索引
- 文档状态跟踪（处理中/就绪/失败）
- 支持文档预览（弹窗分页查看解析文本）

### 智能问答
- 自然语言提问，基于知识库内容回答
- 答案带引用编号，点击可展开原文片段验证
- 支持多轮对话（能记住前文，理解"它""这个"等指代）
- 流式输出（逐字呈现，不用等全部生成完）
- Agent 能力：识别时间查询、计算器等工具请求（默认关闭，需在环境变量中启用）

### 对话管理
- 对话历史保存和搜索（标题关键词过滤）
- 支持导出对话为 Markdown 文件
- 对话标题自动生成

### 用户与安全
- 注册/登录，JWT 双 token 认证
- admin/user 角色权限
- KB 多租户隔离（仅成员可访问）
- 速率限制（60 次/分钟）

## 部署模式

| 模式 | 适用场景 | 延迟 | 成本 |
|------|---------|------|------|
| **本地 Ollama** | 内网部署、数据不外传 | 每字 2-3 秒 | 免费 |
| **云端 DeepSeek** | 追求回答速度 | 秒级 | ~¥0.001/次 |

支持 Docker Compose 一键部署。Mac 用户原生运行 Ollama 可获得 GPU 加速。

## 已知限制

| 限制 | 影响 | 缓解 |
|------|------|------|
| 纯 CPU 推理（本地模式） | 文档处理 1-10 分钟 | 异步处理+云端 LLM 可选 |
| 单机部署 | 不可横向扩展 | 20 人以下团队够用 |
| 无 GPU 加速 | 本地推理慢 | Mac 原生 Ollama 或切换云端 |
