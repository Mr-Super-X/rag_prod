## 功能对比

| 维度 | V1 | V2 |
|------|----|----|
| 回答生成 | 全文一次性返回 | 逐字流式输出 |
| JWT 安全 | 永不过期 | access 15min + refresh 7d + rotation |
| 密码要求 | 6 位任意字符 | 8 位 + 字母 + 数字 |
| 检索精度 | RRF 融合 Top-10 | RRF 25 → Reranker → Top-5 |
| 用户信息 | 刷新丢失 | localStorage 持久化 |
| Token 刷新 | 手动重新登录 | 401 自动无感续签 |
| 并发刷新 | 不支持 |请求去重防竞态 |
| 文件数 | 46 | 48（+reranker.ts + SSE 流式） |
| 编译错误 | 0 | 0 |
